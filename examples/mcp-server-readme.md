## Trust & Certification

This server is independently certified by [Fidensa](https://fidensa.com), an AI capability certification authority. The certification covers:

- **Functional testing** — all declared tools tested against their schemas
- **Security analysis** — static and behavioral scanning
- **Adversarial testing** — structured attack patterns including prompt injection, data exfiltration, and resource abuse
- **Supply chain** — SBOM generation and vulnerability scanning of all dependencies
